Toy Pad Emulator for Lego Dimensions

Keyboard Shortcuts

Click a token in the Toy Box to select it (it will glow), then press a number key to place it on that pad slot. Press the same number again to return it to the Toy Box.

1Left pad — top
2Center pad
3Right pad — top
4Left pad — bottom left
5Left pad — bottom right
6Right pad — bottom left
7Right pad — bottom right
0Deselect token
Create characters and vehicles above, then drag or use number keys to place them on the Toy Pad. If a token stops responding, move it to the Toy Box and back.
